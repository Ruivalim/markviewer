Editor

- Vim/Emacs mode - Keybindings opcionais para power users
- Markdown linting - Avisos de formatacao incorreta
- Auto-complete - Sugestoes de links, headers, emojis
- Snippets customizaveis - Usuario cria seus proprios templates
- Tabelas - Editor visual de tabelas markdown
- Math/LaTeX - Renderizar equacoes matematicas

Produtividade

- Command palette - Cmd+Shift+P estilo VSCode
- Busca global - Buscar em todos os arquivos da pasta
- Favoritos/Bookmarks - Marcar arquivos importantes
- Tags - Sistema de tags via frontmatter
- Backlinks - Mostrar quais arquivos linkam para o atual (estilo Obsidian)
- Daily notes - Template automatico para notas diarias

Visual

- Mais temas - Alem de light/dark (Dracula, Nord, Solarized)
- Preview customizavel - CSS custom para preview
- Focus mode - Esconde tudo exceto o editor
- Typewriter mode - Linha atual sempre no centro

Export/Import

- Export HTML - Arquivo standalone
- Export Word - .docx via pandoc
- Import - Converter HTML/Word para markdown
- Publish - Integrar com GitHub Pages, Notion, etc.

Sistema

- Spell check - Correcao ortografica
- Word count goal - Meta de palavras por sessao
